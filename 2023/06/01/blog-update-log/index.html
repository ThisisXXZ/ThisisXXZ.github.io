Blog Update Logs

Pinned Posted on 2023-06-01 Edited on 2023-06-14 繁/简： set

本博客的更新日志。

version info: hexo:6.3.0, hexo-cli:4.3.1, NexT 8.17.0, node:18.16.0, git:2.39.1.windows.1


Version 1.0

23.5.2023: 本站成功建立！采用 Hexo 搭建静态博客框架并部署至 Github Pages。

24.5.2023: 采用 NexT.Mist 博客主题并进行自定义优化。

  • 增加 about, tags, categories 详情页
  • 增加 meta-post 栏的字符统计和预计阅读时间栏目
  • 设置侧边栏个人信息
  • 改变网站头图 favicon
  • 设置背景 canva_nest 动效
  • 在侧边栏与博文页面展示 CC License BY-NC-SA
  • (侵入式更改) 将 tags 标识符由 # 改为 fa fa-tags
  • (侵入式更改) 增添文章末尾提示
  • (侵入式更改) 点击位于侧边栏的头像 avatar 跳转至网页主界面

25.5.2023: 开始博客迁移计划，将原 CnBlogs 上的部分博文进行转移。

26.5.2023: 更改渲染引擎。

  • 将默认的渲染引擎由 hexo-renderer-marked 更改为 hexo-renderer-kramed 以更好的支持 markdown 数学公式渲染
  • 应用 hexo-asset-image 插件简化图片上传
  • 应用 hexo-generator-index-pin-top 支持文章置顶功能
  • (侵入式更改) 增加 post-meta 栏的文章置顶图标

27.5.2023: 博客功能添加。

  • 增加站内搜索页 search
  • 增加评论功能，采用 valine 评论系统
  • (侵入式更改) 通过注释 js 中的相关代码取消评论数统计显示

28.5.2023: 更改显示字体。

  • (侵入式更改) 仅仅减小 post 中的字体大小
  • 将全局字体由 Lato 改为 Noto Serif SC
  • 将文内标题 (in-post heading) 字体设为 Sitka Heading 且比例为 2

29.5.2023: (侵入式更改) 增加 meta-post 栏的简繁转换功能。

  • issue：该简繁转换功能仅仅建立一个由简体中文字符到繁体中文字符的一对一映射，因此在实际使用时会出现一些问题。例如：简体“表面”一词将会被错误的转化为“表麵”，这是由于程序没有考虑简体与繁体间一对多映射的情况。

1.6.2023: 在 NameSilo 上自购域名 thisisxxz.com (有效期一年) 并将其绑定至 thisisxxz.github.io。配置 Cloudflare 对 thisisxxz.com 进行 CDN 加速，提升网站加载速度。(然而大图加载还是有点慢)

2.6.2023: (侵入式更改) 在侧边栏添加 RSS 订阅源。

3.6.2023: 应用 theme-next/hexo-next-share 插件增加文章分享功能。

  • (侵入式更改)：将分享按钮调整到 CC License 之后

  • issue：该插件似乎会导致执行命令 hexo g 与 hexo s 时弹出警告 WARN: The upstream repository of theme NexT has been changed，但似乎未产生任何实质影响。推测是因为当前 NexT 版本 (7.8.0) 不是最新所致。已提交 discussion。

7.6.2023: (侵入式更改) 在侧边栏添加站点地图 sitemap。将网站 https://www.thisisxxz.com/ 提交至 google 与 bing 搜索引擎进行收录。

  • issue: 由于 GitHub pages 拒绝百度爬虫访问，现时无法将部署在 GitHub pages 上的网站提交至百度进行收录。考虑解决方案：双线部署。 (感觉好麻烦，算了，百度你有眼无珠)


Version 2.0

第一次重大更新！

9.6.2023: (侵入式更改) 在侧边栏添加网易云音乐播放器并播放超好听的三日月之舞。调整 sitemap 与 rss 按钮为平行显示。(有样学样写了一段 css 代码，居然能跑，好开心) 暂时关闭友链显示。

10.6.2023: 将 NexT 主题由 NexT.Mist 更换为 NexT.Pisces。

  • 将 sitemap 与 rss 按钮由侧边栏删除并加入菜单栏
  • 更改网易云音乐播放器样式
  • 启用 pjax 实现音乐全局播放
  • 为 about 详情页添加四个子页面 playlist, anime, games 与 books

10.7.2023: 重大样式更新。

  • (侵入式更改) 增大 post 标题的字体大小，并将其颜色修改为 赤 (#c3272b)
  • (侵入式更改) 将侧边栏高亮字体由橙色修改为 大红 (#ff2121)
  • 移除 meta-post 栏的字符统计和预计阅读时间栏目
  • 将 canvas-nest 动效改为 canvas-ribbon 动效，并增添鼠标点击爆炸特效
  • 增添动态网站标题栏
  • 为配合站内红黑主色调，将头像由黄大叔换成学妹 (我们将永远铭记黄前部长在 v1.0 中所做的巨大贡献)

14.6.2023: (NexT 重大版本更新) 将 NexT 版本由 7.8.0 更新至 latest 8.17.0。基本复现了之前所做的所有侵入式更改。解决了移动端博客文字不显示的问题。仍有两个问题待解决：

  • NeedMoreShare2 插件显示位置出现问题。已提交 issue。
  • (已解决) 新版本取消了对 Valine 的支持。重新加载 Valine 后评论消失。

15.6.2023: (侵入式更改) 在 copyright 栏目中的博客链接末尾增添复制按钮。
